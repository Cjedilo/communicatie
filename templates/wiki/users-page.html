Users

The Users page shows everyone you can interact with — people on your own server and on connected servers.

Browsing users Starting a conversation Viewing a profile Blocking someone Admin actions

Browsing users

Local users appear at the top. Remote users are grouped by their home server and collapsed by default — click a server name to expand the list. Use the filter bar to search across everyone at once.

Starting a conversation

Next to each user you'll find action buttons:

  • 💬 — opens a direct message channel between you and that person. It's a private channel: only the two of you can see it.
  • 🗒 (next to your own name) — opens your Scratchpad, a private channel with yourself. Useful for notes, links, or anything you want to keep handy.

Viewing a profile

Click on a user's name or avatar to open their profile. You'll see their display name, avatar, when they joined, and their recent messages. Clicking a message takes you straight to that channel.

Blocking someone

Next to each user is a 🔕 button. Click it to block that person — their messages will no longer appear in your stream or any channel. The button turns 🚫 when they're blocked. Click again to unblock.

You can also block from someone's profile page.

Admin actions

If you're the server owner, you'll also see a ✕ button next to local users. This permanently deletes their account and all their messages.
